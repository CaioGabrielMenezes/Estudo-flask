# Mini Projeto Flask + Tailwind

Projeto simples desenvolvido para estudar:

- Flask
- Direcionamento de rotas
- Templates
- Integração com Tailwind CSS

---

## Objetivo

Praticar a criação de rotas no Flask, renderização de templates e estilização com Tailwind CSS em uma aplicação web básica.

---

## Tecnologias utilizadas

- Python
- Flask
- HTML
- Tailwind CSS

---

## ⚙️ Funcionalidades

- Navegação entre múltiplas rotas
- Templates dinâmicos
- Layout estilizado com Tailwind
- Estrutura básica de app Flask
